# [转换成小写字母](https://leetcode-cn.com/problems/to-lower-case/)

## 方法：位运算

### 思路与算法

在``ASCII``码中，``A``对应``100,0001``,``Z``对应``101,1010``,``a``对应``110,0001``,``z``对应``111,1010``,可见大小写字母只在二进制第6位上有差别。据此有如下结论：

- 大小写切换：字符^=32;

- 全部转小写：字符|=32;

- 全部转大写：字符&=-32;

### 代码

```c++
class Solution {
public:
    string toLowerCase(string s) {
        for(char& ch:s)
            if(ch>=65&&ch<=90)
                ch|=32;
        return s;
    }
};
```

### 复杂度分析

- **时间复杂度**：$O(N)。

- **空间复杂度**：$O(1)。
